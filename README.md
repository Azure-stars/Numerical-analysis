## 环境配置

根目录下运行：

```bash
$ pip install -r requirements.txt
```

```txt
numpy
scipy
matplotlib
sympy
```



## 运行方式

对于每一个实验，在根目录下运行对应名称的 py 文件即可。如想运行实验  1 的代码，可以运行如下指令：

```bash
$ python3 LAB1.py
```

